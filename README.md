# Simple NFT Minting Tool

This project lets you mint your own NFT.
It works on Ethereum testnet.
You can learn how smart contracts work.

This project is made for beginners.
All code is simple and clean.

## Features
- Mint NFT from command line
- Connect to testnet wallet
- Upload metadata
- View minted token ID

## Requirements
- Node.js installed
- MetaMask wallet
- Testnet ETH
- Free RPC provider account

## Setup
1. Clone this repo
2. Open folder in terminal
3. Run: npm install
4. Create a file called .env
5. Add your private key and RPC URL
6. Run: node mint.js

## Example
Run the script.
The NFT will be minted.
Token ID will be printed.

## Learning Goals
- Learn how NFTs work
- Learn how to deploy and call contracts
- Learn basic blockchain transactions
